# npm-learn

Scanner CLI de seguridad pre-instalación para paquetes npm. Detecta y enseña todo acerca de vulnerabilidades recientes en el ecosistema npm.

Proyecto Final — Desarrollo Profesional 1 (C1335), 2026-1.
Autora: Anaís Polett Rodríguez Carvajal.

## Instalación

```bash
git clone <url-del-repositorio>
cd npm-learn
npm install
npm link
```

## Uso

```bash
npm-learn <nombre-del-paquete>
npm-learn <nombre-del-paquete>@<version>
```

Ejemplo:

```bash
npm-learn reactt
npm-learn lodash@4.17.20
```
